🌞 Android Studioとは？

  • Android Studioは、Androidアプリを作成するための開発ツールです。
  • この章では、 メモアプリの画面（フロントエンド） を作成します。
  • Androidアプリは、ユーザーの操作を受け取り、サーバー（バックエンド）と通信します。

🌞 準備をしよう！

🖊 使用するAndroid Studioのバージョン

  • この教材では Android Studio Otter 2025.2.1 Patch 1 を使って説明します。
  • バージョンが異なる場合、表示や操作手順が一部異なることがあります。

🖊 他のプロジェクトを閉じよう（開いている場合）

  1. 左上のハンバーガーメニュー（≡）をクリックしてください。
  2. File → Close Project をクリックしてください。

🖊 新しいProjectを作成しよう

  1. 「NewProject」をクリックしてください。
  2. 「Empty Views Activity」を選択し、Nextをクリックしてください。
  3. 以下の設定を行い、Finishをクリックしてください。
  • Nameは 「MemoClient」 に変更
  • Minimum SDKは 「API36.0（“Baklava”;Android 16.0）」 を選択
  • Build configuration language は 「Groovy DSL（build.gradle）」 を選択
  1. Gradleのimportが終わるまで待ちます。

🌷 Project「MemoClient」の作成完了！

🌞 フォルダ構成を作ろう

🖊 3つの「entities」「resources」「views」Packageを作成しよう

  1. ➀app → ②java → ③com.example.memoclient の順にクリックしてください。
  2. com.example.memoclient を右クリックしてください。
  3. ➀New → ②package の順にクリックしてください。↓
  4. com.example.memoclientの続きに「entities」を入力してEnterしてください。

🌷 「entities package」の作成完了！ 5. 同様に「resources」「views」packageを作成してください。↓

🌷 3つのpackageの作成完了！

🖊 Java Classを作成しよう

  • views の中に Memo Activity.java を作成します。
  1. com.example.memoclientの中にあるMainActivityを右クリックし、Renameをクリックしてください。
  2. 名前を「MemoActivity」に変更し、Refactorをクリックしてください。
  3. MemoActivityをドラッグして、views packageに入れてください。
  4. Refactorをクリックしてください。
  5. viewsをクリックし、MemoActivityが入っているか確認してください。

🌷 「Memo Activity.java」の作成完了！

  • entities の中に Memo.java を作成します。
  1. entities を右クリックしてください。
  2. ➀New → ②Java Class の順にクリックしてください。
  3. 「Memo」を入力してEnterをクリックしてください。

🌷 「Memo.java」の作成完了!

  • resources の中に MemoResource.java を作成します。
  1. resources を右クリックしてください。
  2. ➀New → ②Java Class の順にクリックしてください。
  3. 「MemoResource」を入力し、「Interface」をクリックしてEnterをクリックしてください。

🌷 「Memo.java」の作成完了!

🌞 メモ入力画面を作成しよう

  • まずは、ユーザーが操作する画面を作成します。
  • 画面はXMLファイルでレイアウトを定義します。

🖊 メモ入力欄を作成しよう

  • EditTextを使って、メモを入力する欄を作成します。
  • ユーザーはここに保存したいメモの内容を入力します。

🖊 保存ボタンを作成しよう

  • Buttonを使って、メモを保存するためのボタンを作成します。
  • このボタンを押すことで、サーバーにデータを送信します。

🖊 メモ表示エリアを作成しよう

  • TextViewを使って、保存されたメモを表示するエリアを作成します。
  • サーバーから取得したメモの内容をここに表示します。

🌞 メモAPIと通信する処理を実装しよう

  • 次に、サーバーと通信する処理を実装します。
  • 通信には、HTTPリクエストを使用します。

🖊 GET /memo を呼び出そう

  • サーバーに保存されているメモを取得する処理を書きます。
  • 取得したデータを、メモ表示エリアに反映します。

🖊 PUT /memo を呼び出そう

  • 入力欄に書いたメモをサーバーに送信する処理を書きます。
  • 正しく送信できた場合、サーバー側でメモが保存されます。

🌞 アプリを実行して動作を確認しよう

  • エミュレータまたは実機でアプリを実行します。
  • メモを入力し、保存ボタンを押します。
  • メモが保存され、再取得できれば成功です。

🌞 Android Studioで作成した内容を整理しよう

  • Android Studioでは、ユーザーが操作する画面を作成しました。
  • ボタン操作をきっかけに、サーバーと通信する仕組みを実装しました。
  • これにより、バックエンドとフロントエンドが連携したメモアプリが完成しました。
